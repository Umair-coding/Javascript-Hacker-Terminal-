# 💻 Hacker Terminal Simulation

A simple **Hacker Terminal Simulation** built using **HTML, CSS, and JavaScript**.

## 📌 About

This project simulates a fake hacker terminal that displays messages one by one with random delays, creating a realistic terminal effect.

It was built as a JavaScript practice project while learning:

* Async / Await
* Promises
* Random Delays
* DOM Manipulation
* Loops
* Timing Functions (`setTimeout` / `setInterval`)

## 🚀 Features

* Green hacker-style terminal
* Random delay (1–7 seconds) between messages
* Animated terminal output
* Clean and beginner-friendly code

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📚 What I Learned

While building this project, I practiced:

* Working with Promises
* Using `async` / `await`
* Creating reusable functions
* Manipulating the DOM
* Generating random delays
* Writing cleaner JavaScript code
